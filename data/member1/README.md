# Member 1 — data

Drop your cleaned dataset here as a **.csv** file (any filename).

Required columns must match your entry in `src/config.py` (member id 1):
- a date column      (default: `date`)
- a category column  (default: `category`)
- one or more metric columns (default: `metric_a`, `metric_b`)

Until a CSV is present, this section runs on generated **sample data**.
